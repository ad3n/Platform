Platform
========

**Platform** adalah sebuah Enterprise Bootstrap yang berisi:

> + Doctrine ORM

> + Doctrine ODM (MongoDB)

> + Centralize Session (Redis)

> + Admin Generator

> + Restful API

> + Api Doc Generator

> + Cache Management

> + Elastic Search

> + RabbitMQ

Semua telah dikonfigurasi sehingga Anda tinggal install dan semua dapat langsung dipakai
